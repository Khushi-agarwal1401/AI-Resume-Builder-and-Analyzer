# **Builders Program**

# **Problem Statement 1**

# **AI Resume Builder & Analyzer**

**Difficulty:** Intermediate

**Recommended Duration:** 4 Sprints (3 Weeks)

---

# **1\. Background**

Recruiters receive hundreds of resumes for every job posting. Most candidates struggle to create resumes that effectively showcase their skills, achievements, and experiences. Additionally, many resumes fail Applicant Tracking System (ATS) screening due to poor formatting, missing keywords, or weak content.

Existing resume builders primarily focus on design and formatting, while users increasingly need intelligent guidance to improve the quality of their resumes.

The objective of this challenge is to design and build an AI-powered Resume Builder and Resume Analyzer that helps users create professional resumes, evaluate them against ATS standards, and receive actionable recommendations for improvement.

---

# **2\. Problem Statement**

Design and develop a web application that enables users to:

1. Build resumes from scratch by providing necessary profile details and JD details   
2. Import existing resumes.  
   1. Analyze resume quality.  
   2. Compare resumes against job descriptions.  
   3. Improve resume content using AI.  
   4. Export professional resumes.

The final solution should be suitable for real-world use and demonstrate good product thinking, scalability, and engineering practices.

---

# **3\. Business Objective**

The solution should aim to:

* Reduce resume creation time.  
* Improve resume quality.  
* Increase ATS compatibility.  
* Help candidates tailor resumes for different jobs.  
* Provide meaningful AI-driven suggestions.

---

# **4\. Target Users**

### **Primary Users**

* Undergraduate students  
* MBA students  
* Fresh graduates  
* Job seekers  
* Early professionals

### **Secondary Users**

* Universities  
* Placement Cells  
* Career Coaches

---

# **5\. Functional Requirements**

The application should support the following capabilities.

## **User Management**

The system should allow users to:

* Register  
* Login  
* Manage their profile  
* Save multiple resumes

---

## **Resume Builder**

Users should be able to create resumes containing:

* Personal Information  
* Education  
* Experience  
* Projects  
* Skills  
* Certifications  
* Achievements  
* Positions of Responsibility  
* Languages  
* Interests

The builder should allow editing, deleting, and rearranging sections.

---

## **Resume Analysis**

Users should be able to upload an existing resume for analysis.

The system should provide insights on:

* ATS compatibility  
* Formatting  
* Missing sections  
* Readability  
* Grammar  
* Resume strength

---

## **Job Description Matching**

Users should be able to upload or paste a Job Description.

The system should compare the resume against the Job Description and identify:

* Missing keywords  
* Skill gaps  
* Relevant experience gaps  
* Overall match percentage

---

## **AI Assistance**

The application should provide AI-assisted features such as:

* Resume summary generation  
* Bullet point enhancement  
* Resume rewriting  
* Achievement suggestions  
* Grammar correction

---

## **Resume Export**

Users should be able to export resumes in printable formats.

---

# **6\. Non-Functional Requirements**

The solution should demonstrate:

* Responsive design  
* Secure authentication  
* Efficient data storage  
* Good application performance  
* Clean architecture  
* Maintainable codebase
